About

This business big data dashboard is a semester-long project which Cornell collaborated with Axtria, a business big data analytics company. The goal is to build a fully-functioning, visually appealing, and user-friendly new user interface for the data dashboard. In addition, we needed to connect the interface with the back-end data set on Amazon Web Service. I designed the user interface and developed the front-end of this data dashboard.Take a look at the prototype!

STEP 1: User Interview

We interviewed a user who works in Axtria as a data analytist. After gaining insights of the user needs and analyzed the requirement of client, we built several version of UI wireframe and went through several iterations before we moved on to the high-fedality prototype.
