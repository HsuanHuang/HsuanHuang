About

United Insights is a collaboration workspace for Credit Suisse. To be more specific, this system is for employees working on Politically Exposed Person (PEP) review, which requires them to work closely with many other departments.

Started from mid-fidelity prototype, I recreated it using Balsamiq to conduct user testing. After finding some critical questions in the prototype, I iterated it into high-fidelity prototype using Sketch. The next steps are making it an interactive prototype using Invision and conducting a final user testing for future iteration.

Persona & User Journey

The persona is named Anthony, who is a compliance officer at Credit Suisse. His daily tasks include many PEP review and on-boarding new client.

Mobile application development is not in the scope of this project.

The user journey clearly defines what client expects the workspace should perform and how the users are going to interact with it. It helps me to identify the user goal and system requirement in good details.

Key Features

After analyzing the user journey and communicating with the client, there are five key features in designing the collaboration workspace:

1. A task overview showing recently due and new coming tasks
2. A task detail page to check its progress
3. A detail view of clients to review their information
4. A collaboration dashboard showing the activities of other collaborators
5. An instant messenger to communicate with collaborators internally

Key Question:

" How to present the tasks clearly and make the collaborators work more efficiently and effectively? "
