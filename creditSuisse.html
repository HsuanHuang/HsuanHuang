Project Details

The project goal is to build an on-line collaboration workspace, United Insights. The client of this project is Credit Suisse. This is an on-going project of mine which is expected to completed in this May.

The workspace should be constructed by integrating existing softwares ,such as Moxtra, Jive, and Skype. I'm working as the UI/UX designer in this project. I will conduct user interview to identify user needs, design the UI wireframe, build a prototype using Balsamiq and InVision, then conduct the user test to iterate the prototype before the project move toward the development phase. The final outcome will be put on in the future.
